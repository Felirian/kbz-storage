# NovaTech Solutions

## О компании

NovaTech Solutions — продуктовая студия, специализирующаяся на разработке веб-приложений для B2B-сегмента. Основная специализация — SaaS-платформы, внутренние корпоративные системы, интеграционные решения.

Компания работает с корпоративными клиентами: внутренние заказчики из числа HR-отделов, бизнес-аналитиков и IT-партнёров получают готовые продуктовые решения, а не только разработку по требованиям.

## Продукты и проекты

## Технологическая база

Студия придерживается единого технологического стека для всех фронтенд-проектов:

- **Фреймворк:** React 18, function components, hooks
- **Язык:** TypeScript (strict mode)
- **Стилизация:** SCSS Modules, BEM-нейминг
- **Стейт-менеджмент:** Zustand (глобальный), React Context (локальный)
- **HTTP:** Axios с кастомными interceptors
- **Тестирование:** Vitest + React Testing Library
- **CI/CD:** GitHub Actions → Vercel (preview) / AWS (production)
- **Дизайн:** Figma, Storybook

Отклонения от общего стека возможны на уровне проекта и фиксируются в RFC (см. соответствующий раздел в каждом проекте).

## Принципы работы

- Каждый проект имеет письменный паспорт, архитектурный документ и регламент работы — документация хранится в этой базе знаний.
- Изменения в архитектуре, технологическом стеке и security-чувствительных областях проходят через RFC с явным апрувом.
- Код-ревью обязательно для всех изменений; джуниоры получают менторские комментарии, не только технические.
- Дизайн и разработка работают в одном спринте; хендофф оформляется по регламенту (см. раздел «Дизайн» каждого проекта).
