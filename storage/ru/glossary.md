# Глоссарий

## База знаний

База знаний — централизованная система хранения документации и накопленного опыта команды.

Пример использования базы знаний представлен в проекте [Aurora](/projects/aurora/passport).

## Паспорт проекта

Паспорт проекта — документ, содержащий основную информацию о проекте: цели, участников, технологический стек и этапы реализации.

Пример паспорта проекта можно увидеть в статье [Паспорт проекта Aurora](/projects/aurora/passport).

## Техническое задание

Техническое задание — документ, описывающий функциональные и нефункциональные требования к системе.

Подробный пример приведен в статье [Техническое задание проекта Aurora](/projects/aurora/requirements).
