# Роли участников проекта

## Общая модель ролей

В проекте Aurora используется ролевая модель управления доступом.

Описание проекта и его целей приведено в документе [Паспорт проекта](../aurora-project-passport.md).

## Product Owner

Product Owner отвечает за:

- формирование продуктового видения;
- управление бэклогом;
- постановку задач команде разработки.

## Project Manager

Project Manager контролирует:

- сроки выполнения задач;
- распределение ресурсов;
- коммуникацию между участниками проекта.

## Разработчики

Разработчики реализуют программную логику системы.

Архитектурные решения, используемые в проекте, описаны в статье [Архитектура системы Aurora](../technical/aurora-architecture.md).

## Дизайнеры

Дизайнеры отвечают за:

- пользовательский интерфейс;
- дизайн-систему;
- взаимодействие пользователя с системой.

Рекомендации по оформлению документации для дизайнеров можно найти в статье [Передача макетов в разработку](../../design/handoff-guidelines.md).
