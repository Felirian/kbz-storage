# Стайлгайд Forge

## Назначение

Стайлгайд фиксирует визуальный язык Forge. У продукта две аудитории — конструкторы и исполнители — и стайлгайд служит обеим, при этом давая каждой свой набор акцентов.

## Цветовая палитра

### Основные

| Токен | HEX | Назначение |
|---|---|---|
| `--color-primary-500` | `#0F766E` | основной акцент, активные действия |
| `--color-primary-600` | `#0B5E58` | hover на primary |
| `--color-primary-100` | `#E0F2F1` | фоны выделенных областей |
| `--color-secondary-500` | `#9333EA` | акцент конструктора (поля в режиме редактирования) |
| `--color-accent-500` | `#F97316` | акценты в маршрутах согласования |

### Нейтральные

| Токен | HEX |
|---|---|
| `--color-neutral-900` | `#111827` |
| `--color-neutral-700` | `#374151` |
| `--color-neutral-500` | `#6B7280` |
| `--color-neutral-300` | `#D1D5DB` |
| `--color-neutral-100` | `#F3F4F6` |

### Семантические

| Токен | HEX | Назначение |
|---|---|---|
| `--color-success-500` | `#16A34A` | успешные действия, шаг согласован |
| `--color-warning-500` | `#EAB308` | требует внимания |
| `--color-danger-500` | `#DC2626` | отказ, ошибка валидации |
| `--color-info-500` | `#0284C7` | информационные сообщения |

Цвета доступны как CSS-переменные (`app/styles/tokens.css`) и как объект `tokens.color` в TypeScript.

> ⚠️ Внимание: режимы редактора и рантайма используют разные акцентные цвета. В редакторе — `--color-secondary-500` (фиолетовый) для подсветки активного поля. В рантайме — `--color-primary-500` (бирюзовый) для CTA. Не путайте контексты.

## Типографика

Основной шрифт — **Inter** (variable). Запасной — `system-ui, sans-serif`.

| Токен | Размер | Line-height | Применение |
|---|---|---|---|
| `--text-display` | 32px | 40px | hero на лендинге |
| `--text-h1` | 24px | 32px | заголовки страниц |
| `--text-h2` | 20px | 28px | заголовки секций |
| `--text-h3` | 16px | 24px | заголовки полей и групп |
| `--text-body` | 14px | 20px | основной текст |
| `--text-caption` | 12px | 16px | подписи, метаданные |
| `--text-mono` | 13px | 20px | имена идентификаторов в редакторе (JetBrains Mono) |

Веса: 400 (regular), 500 (medium), 600 (semibold). 700 — только в display.

## Сетка и отступы

- Контентная область — 12 колонок, gutter 24px.
- Максимальная ширина:
  - Лендинг и публичные — 1200px;
  - Редактор — на всю доступную ширину (без max-width);
  - Рантайм — 720px (форма чтения).
- Базовая единица отступов — 4px. Шкала: 4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64.
- Радиусы: `--radius-sm: 4px`, `--radius-md: 6px`, `--radius-lg: 10px`, `--radius-pill: 999px`.

## Тени

| Токен | Назначение |
|---|---|
| `--shadow-sm` | бордеры карточек |
| `--shadow-md` | поповеры, dropdown'ы |
| `--shadow-lg` | модалки, drawer |
| `--shadow-drag` | специальный shadow для перетаскиваемого элемента в редакторе |

Z-index — через токены: `--z-dropdown: 100`, `--z-overlay: 800`, `--z-modal: 1000`, `--z-toast: 1100`. Прямые числа запрещены.

## Иконки

Используется набор Lucide. Размеры — кратные 16px (16, 20, 24). Для специфики Forge добавлены кастомные иконки конструктора (`field-text`, `field-select`, `route-step`, `route-fork`) — в наборе `forge-icons`.

## Принципы применения

- В одном экране — один уровень акцента (primary). Если нужно дополнительное действие — secondary.
- В редакторе фиолетовый используется только для маркеров активного состояния.
- Семантические цвета не используются декоративно.
- При изменении цветов токенов всегда проверяется контраст по WCAG AA.
