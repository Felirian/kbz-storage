# Pulse

SaaS-платформа для HR-аналитики. Дашборды с метриками по сотрудникам, текучести и вовлечённости в режиме реального времени. Статус: **production**, активная поддержка.

**Команда:** [Кирилл](/team#кирилл), [Алексей](/team#алексей), [Олег](/team#олег), [Сергей](/team#сергей)

---

## Содержание

### Менеджмент

- [Паспорт проекта](/projects/pulse/passport) — цель, заказчик, бизнес-метрики, ключевые даты и риски
- [Структура команды](/projects/pulse/team-structure) — роли, зоны ответственности, матрица RACI, каналы связи
- [Процессы](/projects/pulse/processes) — спринт-цикл, ритуалы, Definition of Done, управление инцидентами
- [Беклог и roadmap](/projects/pulse/backlog-roadmap) — приоритеты по RICE, текущие эпики, roadmap по кварталам

### Разработка

- [Используемые технологии](/projects/pulse/tech-stack) — стек (React 18, TypeScript, Recharts, React Query, WebSocket), версии, обоснование выборов
- [Архитектура приложения](/projects/pulse/architecture) — FSD-структура, паттерны Container/Presenter, управление состоянием, RFC-процесс
- [Код-стайл](/projects/pulse/code-style) — нейминг, ESLint-конфиг, Prettier, TypeScript strict, правила по React
- [Работа с репозиторием](/projects/pulse/repository-workflow) — GitFlow, Conventional Commits, правила PR, чек-лист ревью, CI-пайплайн
- [API-взаимодействие](/projects/pulse/api-integration) — REST и WebSocket, авторизация, обработка ошибок, idempotency, мониторинг

### Дизайн

- [Стайлгайд](/projects/pulse/styleguide) — цветовая палитра, типографика, сетка, отступы, иконки

- [Регламент передачи макетов](/projects/pulse/handoff-regulation) — чек-лист хендоффа, критерии приёмки, дизайн-ревью кода
- [UI-библиотека](/projects/pulse/ui-library) — реестр компонентов со статусами (stable/beta/deprecated) и ссылками на Storybook
