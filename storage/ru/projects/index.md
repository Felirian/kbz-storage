# Проекты NovaTech Solutions

В работе три продукта. У каждого — выделенная кросс-функциональная команда и полный комплект документации.

---

## Содержание

- [Pulse](/projects/pulse/index) — SaaS-дашборд для HR-аналитики · статус: **production**
- [Forge](/projects/forge/index) — low-code конструктор форм и документооборота · статус: **beta**
- [Bridge](/projects/bridge/index) — портал для API-партнёров · статус: **pre-launch**

---

## Pulse

SaaS-платформа для HR-отделов. Метрики по сотрудникам, текучести и вовлечённости, real-time обновления через WebSocket. Используется корпоративными клиентами на платной подписке.

**Команда:** [Кирилл](/team#кирилл), [Алексей](/team#алексей), [Олег](/team#олег), [Сергей](/team#сергей)

→ [Документация Pulse](/projects/pulse/index)

---

## Forge

Инструмент для корпоративных клиентов: создание форм, маршрутов согласования и шаблонов документов без кода. В активной разработке, три пилотных клиента.

**Команда:** [Марина](/team#марина), [Дарья](/team#дарья), [Виктория](/team#виктория), [Наталья](/team#наталья)

→ [Документация Forge](/projects/forge/index)

---

## Bridge

Внешний портал для разработчиков-партнёров: документация API, sandbox, управление ключами и квотами, OAuth 2.0 с PKCE. Пилотное тестирование с первыми партнёрами.

**Команда:** [Андрей](/team#андрей), [Марина](/team#марина), [Кирилл](/team#кирилл), [Анна](/team#анна)

→ [Документация Bridge](/projects/bridge/index)
