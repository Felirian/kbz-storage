# Wiki

Welcome to the **kbz-test** project.

## Prerequisites

- Node.js 18+
- pnpm

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Adding New Pages

Simply create a new `.md` file inside the `md/` folder. The filename and folder structure will determine the page URL automatically.
